Technical Blog Part 1

  • Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

    • HTML
      • HTML is what the materials that make up a house are to a website. Everything that is required to build a house is there but without a plan and someone to put them together it is just a pile of materials.
    • CSS
      • CSS is the plan for the house it determines where something should go and what color the walls should be.
    • DOM
      • DOM is the building manager it organises the workers to put the materials where they need to be and states when something doesn't meet requirments.

  • What is meant by boxifying design?

    • Everything on a webpage is inside of a box the nav bar the title the banner image the body and the footer are all inside of one over all box. each section is then made up of its own box which maybe even contain smaller boxes again. even if something doesn't look like a box if you inspect the element you will find that it is indeed a box.

  • What is the box model?

    • The CSS box model refers to the box that wraps around all HTML elements. it is made up of...

    • Content - The content of the box, where text and images appear
    • Padding - Clears an area around the content. The padding is transparent
    • Border - A border that goes around the padding and content
    • Margin - Clears an area outside the border. The margin is transparent/li>

    The box model allows us to do many things such as add borders around elements and define space between elements.
