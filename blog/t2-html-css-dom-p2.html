    • What is grid based design?

      • Grid based design is the concept that a design can be made up of a grid system. grid is not just about squares but about proportions.

    • Whats all the hype about responsive webpages?

      • With the growth of the internet and the rise of smart devices there is a need for websites to be able to be viewed on a host of different devices with varying screen sizes. in order to meet this demand websites need to be able to respond to different screen sizes and display a website that best meets the devices screen.

    • What is semantic structure?

      • Semantic HTML structure is the idea that HTML should convey the underlying meaning of the conent and not it's appearance.
