  • In the JavaScript language, describe all the use cases for each of:
  • parentheses ()

    • Parentheses in programming are generally used to hold an equation or a test. For example they can be used to test is something is true or false, however they can also be used to pass a information in to a sort of sub program or function.


  • brackets []

    • Square brackets are most commonly used to hold indexing information generally a number such as 1 or 2.


  • braces {}

    • curly braces are used to either indicate the start of an object (a piece of information) or they are used to indicate the start of a function (a sort of sub program running inside the main program.)


  • single quotes ' '

    • quotes are used to change the way that Java script reads a piece of information. If we had a variable inside some quotation marks for example 'first' JS would understand that this is a string of letters however if we left it out of the quotation marks then JS would read it as a variable.


  • double quotes " "

    • There is no functional difference between single and double quotation marks however by having the option to use either one we can nest them inside of each other when required.
