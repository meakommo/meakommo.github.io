Backup Configuration of Smart Modem

Use the "Backup" tool to save the Smart Modem's current configuration to a file named "SmartModem_backup.cfg" on your PC. You can then use the "Restore" tool to restore the saved configuration to the Smart Modem. Alternatively, you can use the "Restore to Factory Defaults" tool to force the Smart Modem to perform a power reset and restore the original factory settings.
